# CarDerby

*By Boilse*

Test car derby game

Developed with Unreal Engine 4

Game controls:

          W - move forward
          S - move backward
          A - turn left
          D - turn right

          Mouse wheel up - zoom out
          Mouse wheel down - zoom in
          Mouse middle button - reset zoom

          Esc - Open menu(game pause)
